Memory Hierarchy Diagram

Registers (Fastest, Smallest Size, Inside CPU)
Cache (L1, L2, L3 - Stores Frequent Data for CPU)
RAM (Main Memory, Temporary Storage for Programs)
SSD (Fast Non-Volatile Storage, Used for OS & Apps)
HDD (Slowest, Largest Capacity, Long-Term Storage)

Registers: Smallest and fastest memory, directly inside CPU.

Cache: Stores frequently used data for quick CPU access.

RAM: Holds active processes and running applications.

SSD: Faster than HDD, used for primary system storage.

HDD: Slowest storage, but offers large capacity for files.
